# FUTURE_PE_01 – AI Website Copy Generator for Local Businesses

## Project Overview

This project was created as part of the Future Interns Prompt Engineering Internship.

The goal of this task was to design structured AI prompts that generate high-quality website copy for local businesses.

The project focuses on creating conversion-focused, website-ready content using Prompt Engineering and AI tools.

## Business Chosen

GlowUp Beauty Salon, Mumbai

## What This Project Includes

* Homepage website copy
* Services page content
* Call-to-action (CTA) sections
* Conversion-focused website content
* Brand tone adaptation

## Tools Used

* ChatGPT
* GitHub

## Skills Demonstrated

* Prompt Engineering
* AI Website Copywriting
* Brand Messaging
* CTA Writing
* Website Content Strategy
* AI Content Generation

## Outcome

This project demonstrates how AI can be used to create professional, conversion-focused website copy for real local businesses using structured prompts and reusable workflows.
